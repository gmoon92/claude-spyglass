# Web UI/UX 전반 개선 계획

> 작성일: 2026-04-18  
> Feature: web-ui-ux  
> 대상: packages/web/index.html (단일 파일 대시보드)

---

## 배경 및 목적

Claude Spyglass 웹 대시보드의 UI/UX를 전면 개선한다.
현재 기능 추가가 누적되면서 레이아웃이 복잡해졌고, 디자인 일관성과 제품 완성도가 낮다.

---

## 현재 문제점

### 1. 새로고침 버튼 불필요
- SSE(Server-Sent Events)로 실시간 데이터를 수신 중
- `new_request` 이벤트 → 300ms 디바운스 → fetchDashboard + fetchRequests 자동 실행
- 헤더에 새로고침 버튼이 존재할 이유 없음 → **제거 대상**

### 2. 세션 상세 진입 시 레이아웃 확장 문제
- 세션 ID 클릭 → `#detailSection`이 페이지 하단에 나타남 + `scrollIntoView()` 호출
- 기존 레이아웃 아래로 새 섹션이 추가되어 "확장"되는 UX
- 원하는 UX: 동일한 레이아웃 내에서 패널 전환 또는 슬라이드 인

### 3. 최근 요청 탭 레이아웃 문제
- 최근 요청(Recent Requests) 섹션이 하단에 별도 테이블로 배치
- 브라우저 그리드(프로젝트/세션 선택)와 세션 상세 사이에 불명확한 계층 관계
- 전체 페이지가 수직으로 길어져 스크롤이 과도하게 발생

### 4. 일반적인 디자인 → 제품 가치 하락
- 기본 CSS 변수 + 회색 계열 배경의 평범한 스타일
- 차트, 카드, 테이블 모두 비슷한 밀도와 스타일
- Claude / AI 모니터링 도구로서의 아이덴티티가 없음
- 불필요하게 배치된 섹션(도구 사용 통계 테이블이 항상 보임)

## 현재 레이아웃 구조

```
Header (로고 + LIVE 배지 + 마지막 업데이트 + 새로고침 버튼)
├── Summary Cards (4개: 총 세션, 총 요청, 총 토큰, 활성 세션)
├── Charts Row (타임라인 2/3 + 도넛 1/3)
├── Tool Stats Table (도구별 사용 통계)
├── Browser Grid (프로젝트 320px 고정 | 세션 목록)
├── Detail Section (세션 ID 클릭 시 나타남 → scrollIntoView)
│   ├── 탭: 플랫 뷰 / 턴 뷰
│   └── 요청 테이블 or 아코디언
└── Recent Requests (최근 10개 요청 테이블)
```

---

## 목표 레이아웃 (개선안)

```
Header (로고 + LIVE 배지 + 연결 상태만)
├── Top Strip: Summary Cards (컴팩트, 1줄)
├── Main Layout (고정 높이, 스크롤 없음)
│   ├── Left Panel (280px): 프로젝트 트리 + 세션 목록
│   └── Right Panel (flex-1): 
│       ├── 세션 미선택: 타임라인 차트 + 최근 요청 피드
│       └── 세션 선택: 세션 상세 (플랫/턴 뷰, 슬라이드 인)
└── Footer (최소화)
```

## 기술 스택

- **마크업**: HTML5 (단일 파일)
- **스타일**: 바닐라 CSS (CSS 변수 기반)
- **스크립트**: 바닐라 JavaScript (ES2020+)
- **차트**: Canvas 2D API
- **실시간**: EventSource (SSE)
- **외부 의존성 없음** (CDN 불가 환경 고려)

---

## 개선 범위

### 필수 (Must)
1. **새로고침 버튼 제거** — SSE로 대체됨
2. **세션 상세 진입 UX 개선** — scrollIntoView 제거, 우측 패널 교체 방식
3. **레이아웃 재구성** — 수직 스크롤 → 고정 높이 2패널 구조
4. **디자인 개선** — 색상, 타이포그래피, 밀도 정리

### 선택 (Nice to have)
5. **Tool Stats 섹션 조건부 표시** — 데이터 있을 때만
6. **Recent Requests 피드 통합** — 우측 패널 기본 뷰로 이동
7. **연결 상태 인디케이터 개선** — LIVE 배지를 더 의미 있게
