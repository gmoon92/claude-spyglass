# UX 반복 고도화 Round 3 — ADR

> 작성일: 2026-04-18

---

## ADR-R3-001: 세션 미리보기 XML/태그 제거

**상태**: 결정됨

**문제**: `<task-notification>` 같은 시스템 XML 메시지가 미리보기로 노출됨.

**결정**: `extractFirstPrompt()`에서 HTML/XML 태그 제거 후 텍스트 추출.

---

## ADR-R3-002: 요청 테이블 시각 컬럼 한 줄 표시

**상태**: 결정됨

**문제**: `fmtTimestamp()`의 `<br>` 두 줄 표시가 행 높이를 늘려 데이터 밀도 저하.

**결정**: `"오후 12:21 · 방금"` 한 줄 형식으로 변경. 시·분만 표시(초 제거).

---

## ADR-R3-003: 툴 통계 패널 맥락 명시

**상태**: 결정됨

**문제**: 툴 통계가 선택된 프로젝트와 무관한 전역 데이터임을 UI에서 알 수 없음.

**결정**: 패널 레이블 "툴 통계" → "툴 통계 (전체)" 로 변경.

---

## ADR-R3-004: 세션 목록 행 최소 높이 고정

**상태**: 결정됨

**문제**: 미리보기가 있는 행과 없는 행 높이 불균일.

**결정**: 세션 행 `<td>`에 `min-height` 적용해 시각적 일관성 확보.
