# Favicon 반복 설계 검토 기록

---

## V1 검토 (2026-04-18)

**파일**: `favicon-v1.svg`  
**형태**: 렌즈 원(cx=13,cy=13,r=7.5) + 오른쪽하단 대각선 손잡이  
**배경**: 반투명 원형(fill-opacity=0.88)

### 검토 항목

| 항목 | 평가 | 비고 |
|------|------|------|
| 32px 가독성 | ✅ 통과 | 돋보기 형태 명확히 인식 |
| 16px 가독성 | ⚠️ 보류 | 손잡이 대각선이 너무 얇아 소실 위험 |
| 브랜드 일관성 | ✅ 통과 | accent #d97757 충분히 사용 |
| 다크탭 가시성 | ✅ 통과 | 원형 배경이 캡슐 역할 |
| 라이트탭 가시성 | ⚠️ 보류 | 검은 원이 라이트 탭에서 이질감 |

### 판정: 보류 → V2로 개선

**개선 방향**: 16px 가독성을 위해 조준경(reticle) 형태로 전환. 원형 배경을 둥근 사각으로 변경하여 라이트 탭 이질감 완화.

---

## V2 검토 (2026-04-18)

**파일**: `favicon-v2.svg`  
**형태**: 외곽 링(r=11) + 내곽 링(r=7.5, opacity=0.45) + 크로스헤어 4선 + 중앙 점  
**배경**: 둥근 사각(rx=6, fill=#0f0f0f)

### 검토 항목

| 항목 | 평가 | 비고 |
|------|------|------|
| 32px 가독성 | ✅ 통과 | 조준경 컨셉 즉각 인식 |
| 16px 가독성 | ✅ 통과 | 원+크로스 구조가 소형에서도 선명 |
| 브랜드 일관성 | ✅ 통과 | 모니터링/감시 컨셉 강화 |
| 다크탭 가시성 | ✅ 통과 | 개선됨 |
| 라이트탭 가시성 | ⚠️ 보류 | 여전히 다크 배경 고정으로 라이트 OS에서 어색 |

### 판정: 조건부 통과 → V3에서 라이트 모드 대응 마무리

**개선 방향**: `@media (prefers-color-scheme: light)` SVG 내 미디어쿼리 추가. 중앙 점 크기 미세 조정(r=2 → r=2.5)으로 소형 가독성 강화.

---
