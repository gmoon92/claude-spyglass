# 🔭 Claude Spyglass

**Claude가 어떻게 생각하고 행동하는지, 지금 바로 들여다보세요.**

Claude Spyglass는 Claude Code CLI의 실행 과정을 실시간으로 관찰하고 추적하는 로컬 개발 도구입니다. 프롬프트가 어떻게 처리되는지, 어떤 도구가 호출되는지, 응답이 어떤 과정을 거쳐 생성되는지 한눈에 확인할 수 있습니다.

> **AI는 더 이상 블랙박스가 아닙니다.**

---

## ✨ 왜 Spyglass가 필요한가요?

| 문제 | 해결 |
|------|------|
| AI의 응답은 보이지만 과정은 보이지 않나요? | 프롬프트부터 도구 호출까지 전 과정을 가시화해드립니다. |
| 시스템 프롬프트가 제대로 적용되었는지 확인하기 어려우신가요? | 요청/응답을 구조화하여 검증할 수 있게 해드립니다. |
| 어떤 도구가 언제 호출되는지 파악하고 싶으신가요? | 도구 호출 순서와 입력값을 실시간으로 추적해드립니다. |

---

## 🚀 핵심 기능

- 🔍 **실시간 요청 캡처**
  모든 프롬프트, 응답, 시스템 지시사항을 프록시를 통해 실시간으로 포착합니다.

- 🧠 **도구 호출 추적**
  어떤 도구가 호출되었는지, 어떤 순서로, 어떤 입력값으로 실행되었는지 추적합니다.

- 🌊 **스트리밍 관찰**
  SSE를 통해 응답이 생성되는 과정을 실시간으로 확인합니다.

- 📊 **토큰 & 지연 시간 측정**
  요청별 토큰 사용량과 응답 시간을 정확히 측정합니다.

- 🌳 **트레이스 뷰**
  실행 흐름을 구조화된 트레이스로 시각화합니다 (프롬프트 → 도구 호출 → 응답).

- 🧾 **구조화된 로그**
  모든 이벤트를 JSON 형식으로 저장하여 필터링과 분석이 용이합니다.

## 🌟 비전

AI는 더 이상 블랙박스가 아닙니다.

Claude Spyglass는 LLM의 런타임 동작에 대한 가시성을 제공합니다. AI가 "무엇을" 말하는지뿐만 아니라 "어떻게" 생각하고 행동하는지 이해할 수 있게 해드립니다.
